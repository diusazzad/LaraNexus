# Security Policy for LaraNexus
First of all, thank you for using LaraNexus and helping us keep this small project safe. We take code security seriously and ask that you report any issues you find.
## Supported Versions
Currently, we only provide security patches or updates to the following versions:
| Version | Supported |
| ------- | ------------------ |
| v1.0.x | ✅ Yes |
| < v1.0 | ❌ No |
## Reporting a Vulnerability
If you find a security vulnerability in this project, please do not post it directly as a GitHub Issue publicly. This could give hackers a chance to exploit it.
Instead, send us an message at:
👉 **@sazzadhossainmahim** -- telegram
We will try to respond and resolve it within **48 hours** of receiving your message. We will keep your report confidential.
## Our Promise
We believe that with the help of the community, we can make Laranex even safer. We appreciate your valuable feedback.
